Flex Direction

The flex-direction property determines the direction in which the flex container's items are placed in the flex container.

By default, it is set to row, which means items are placed from left to right.

  • row: Places items in a horizontal row (default).
  • row-reverse: Places items in a horizontal row but in reverse order.
  • column: Places items in a vertical column from top to bottom.
  • column-reverse: Places items in a vertical column but in reverse order.
.container { display: flex; flex-direction: row; }
Flex Wrap

The flex-wrap property controls whether the flex items should wrap onto multiple lines.

By default, items are placed in a single line and will overflow if necessary.

  • nowrap: All items are placed in one line (default).
  • wrap: Items wrap onto the next line when necessary.
.container { display: flex; flex-wrap: wrap; }
Justify Content

The justify-content property aligns flex items along the main axis (horizontal by default).

It is used to distribute extra space between items or align items within the container.

.container { display: flex; justify-content: space-between; }
Align Items

The align-items property aligns flex items along the cross axis (vertical by default).

This property is useful for setting the vertical alignment of items within the container.

.container { display: flex; align-items: center; }
Gap

The gap property defines the spacing between flex items. It can be used to create consistent spacing in a flex container.

You can also specify row-gap and column-gap for more granular control.

  • gap: Sets the spacing between rows and columns.
  • row-gap: Sets the spacing between rows only.
  • column-gap: Sets the spacing between columns only.
.container { display: flex; gap: 10px; }
